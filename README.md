## Getting started

### Clone repository

```bash
$ git clone https://github.com/jeferson-sb/portfolio.git
```

### Install dependencies

```sh
$ cd portfolio
$ npm i
```

### Start local dev server

```sh
$ npm run dev
```

### Linting

```sh
$ npm run lint
```

### Formatting

```sh
$ npm run prettier
```
